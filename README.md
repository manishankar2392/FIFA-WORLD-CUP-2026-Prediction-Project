## 📌 Project Overview

This project is a complete **FIFA World Cup 2026 Analytics Dashboard** developed using **Microsoft Power BI**. It combines historical FIFA World Cup statistics (2006–2022), live tournament analytics, and Round of 32 qualification predictions into one interactive Business Intelligence solution.

The dashboard demonstrates end-to-end data analytics skills including:

- Data Cleaning
- ETL using Power Query
- Data Modeling
- DAX Calculations
- KPI Development
- Interactive Dashboard Design
- Data Visualization
- Sports Analytics

The objective is to transform raw football statistics into meaningful insights that help analyze team performance, player statistics, historical trends, and tournament progression.
